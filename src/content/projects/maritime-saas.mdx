## Overview

Developed a comprehensive SaaS platform for maritime fleet operators to track vessels, optimize routes, and manage maintenance schedules, serving as an internal tool that later became a commercial product.

## Problem Statement

Fleet operators struggled with:
- Fragmented data across multiple systems
- No real-time visibility into vessel locations
- Manual route planning costing 5+ hours weekly
- Reactive rather than predictive maintenance

## Solution Approach

Created an all-in-one platform featuring:
1. Real-time vessel tracking via AIS integration
2. AI-powered route optimization
3. Predictive maintenance scheduling
4. Fuel consumption analytics

## Technical Implementation

**Frontend Architecture:**
- Next.js 14 with App Router
- Real-time updates via WebSocket
- Interactive Mapbox GL maps
- Responsive design (mobile-first)

**Backend Services:**
- PostgreSQL for relational data
- Redis for real-time tracking cache
- Background jobs for route calculations
- RESTful API + GraphQL endpoints

**Key Features:**
- **Live tracking:** Updates every 60 seconds
- **Route optimization:** Considers weather, fuel costs, traffic
- **Maintenance alerts:** ML-based failure prediction
- **Analytics dashboard:** Custom reports and KPIs

## Results & Impact

- **Launched to 5 fleet operators** (150+ vessels)
- **20% fuel cost reduction** through optimized routing
- **40% decrease** in unplanned maintenance
- **$200K ARR** in first 6 months post-launch
